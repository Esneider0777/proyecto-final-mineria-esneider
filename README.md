 Descripción del Proyecto
Este proyecto consiste en entrenar un modelo de Machine Learning capaz de predecir la especie de una flor Iris utilizando 4 características:

- Largo del sépalo  
- Ancho del sépalo  
- Largo del pétalo  
- Ancho del pétalo  

El proyecto también incluye un *dashboard interactivo* hecho con Streamlit, donde el usuario puede ingresar los valores de la flor y obtener una predicción del modelo.

---
Modelo Utilizado
Se utilizó el algoritmo *Random Forest*, debido a que:

- Es un modelo estable y preciso  
- Funciona muy bien con datos pequeños  
- No requiere mucha configuración  
- Es ideal para clasificación  

El dataset Iris proviene de sklearn.datasets.
